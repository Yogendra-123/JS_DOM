Q. 6 What is the difference between window vs document?

Ans. window --> The window object is the topmost object of the DOM hierarchy. It represents a browser window or frame that displays the contents of the webpage. Whenever a window appears on the screen to display the contents of the document, the window object is created.
document --> The document object represent a web page that is loaded in the browser. By accessing the document object, we can access the element in the HTML page.
